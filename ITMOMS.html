  4. HTML – таблиці, форми.
  5. HTML – button … , DOM, заключення.
  6. CSS – введення, пріорітети, підключення
  7. CSS – animate, transform, transition…
  8. CSS – before, after, … заключення.
  9. PHP – Основи синтаксису, особливості використання мови РНР, змінні, константи і оператори.
  10. PHP – типи даних, цикли, умови.
  11. PHP – регулярні вирази, … заключення.
  12. SQL – База даних, основи синтаксису, особливості використання.
  13. SQL – основні операції, … заключення.
  14. Встановлення локального сервера Open server (Аpache, sql, PHPMyAdmin) та налаштування безкоштовного хостінгу (zzz.com.ua).
  15. MVC, встановлення та налаштування WP, адмін панель.
  16. WP – блог, сторінки, плагіни, віджети.
  17. Налаштування WP – JS (синтаксис, основи, підключення).
  18. JS – синтаксис, основи, приклади.
  19. Jquery, Ajax, відправка пошти, завантаження файлів.
  20. Практичне заняття, закріплення матеріалу, знайомство з особливостями використання браузерів.
  21. Вибір фінального проекту, практичне заняття, слайдери, форми, сайдбари.
  22. Практичне заняття, огляд типів побудови сторінок WP (шорт-коди, шаблони).
  23. Практичне заняття, особливості обробки асинхронних даних, довантаження контенту JS.
  24. Захист фінального проекту.
